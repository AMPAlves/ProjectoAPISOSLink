# Projecto API -- SOSLink

API Endpoints:
- [GET] Homepage -> <b>/</b> 
- API Request [POST] SignIn -> <b> users/SignIn </b>
    - username -> Name do User
    - password -> Password do User 
- API Request [POST] SignUp -> <b> users/SignUp </b>
    - username -> Name do User
    - password -> Password do User
    - cpassword -> Confirm Password
    - email -> Email do User
