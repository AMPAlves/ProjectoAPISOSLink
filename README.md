# Projecto API -- SOSLink

API Endpoints:
- [GET] Homepage -> / 
- API Request [POST] SignIn -> users/SignIn
    Fields SignIn:
    - username -> Name do User
    - password -> Password do User 
- API Request [POST] SignUp -> users/SignUp
     Fields SignIn:
      - username -> Name do User
      - password -> Password do User
      - cpassword -> Confirm Password
      - email -> Email do User
